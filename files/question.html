Question-1: What are the different ways to select an element in the DOM?

Selects a single element by getElementById(), Selects multiple elements with the same class name by getElementsByClassName(), Selects all elements with a specific tag name by getElementsByTagName(), Selects the first element that matches a CSS selector by querySelector(), Selects all elements that match a CSS selector by querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML দিয়ে ডাইনামিকভাবে HTML কনটেন্ট যোগ বা পরিবর্তন করতে যায় , innerText ব্যবহার করে text দেখা যায় এবং সেটও করা যায় , textContent ব্যবহার করে সব টেক্সট পাওয়া যায়, লুকানো এলিমেন্ট সহ এবং স্পেসও ।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হলো একটি কৌশল যেখানে একটি প্যারেন্ট এলিমেন্ট তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট হ্যান্ডেল করতে পারে, এমনকি যদি সেই চাইল্ড এলিমেন্টগুলো ডায়নামিকভাবে যোগ করা হয়। এটি পারফরম্যান্স উন্নত করে এবং কোডকে আরও সহজভাবে ব্যবস্থাপনা করতে সাহায্য করে।

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object (Provided the handler is initialized).

Question-5: How do you create, add, and remove elements using JavaScript?

Create a new HTML element using the document.createElement() method. Add elements using appendChild(), append(), prepend(), insertBefore(), etc. Remove elements using removeChild() or remove().
